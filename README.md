# 📚 Sistema de Biblioteca em Java

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar os conceitos de Programação Orientada a Objetos (POO) em Java.

O sistema simula uma biblioteca simples, permitindo o gerenciamento de livros, empréstimos e devoluções.

---

## 🚀 Funcionalidades

* ✅ Listar livros disponíveis
* ✅ Buscar livro por título ou autor
* ✅ Realizar empréstimo de livro
* ✅ Devolver livro
* ✅ Tratamento de exceções

---

## 📂 Estrutura do projeto

```
Mini_Projeto_Biblioteca
 ├── app

 ├── model

 │    └── Professor.java

 │    └── Emprestimo.java
```
